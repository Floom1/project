# HR-Сервис — система автоматизации отдела кадров

Веб-приложение на Django для управления вакансиями, кандидатами, документооборотом и аналитикой найма.

## Требования

- Python 3.14+
- [uv](https://docs.astral.sh/uv/) — менеджер пакетов

## Установка и запуск

```bash
# 1. Клонировать репозиторий и перейти в папку проекта
cd project

# 2. Установить зависимости
uv sync

# 3. Применить миграции
uv run python hr_service/manage.py migrate --pythonpath hr_service

# 4. Создать суперпользователя (роль admin)
uv run python hr_service/manage.py createsuperuser --pythonpath hr_service

# 5. (Опционально) Заполнить БД тестовыми данными
uv run python hr_service/manage.py seed --pythonpath hr_service

# 6. Запустить сервер
uv run python hr_service/manage.py runserver --pythonpath hr_service
```

Приложение будет доступно по адресу http://127.0.0.1:8000/

## Структура модулей

| Модуль       | Django-приложение | Описание                                         |
|--------------|-------------------|--------------------------------------------------|
| Accounts     | `accounts`        | Пользователи, роли, авторизация                  |
| Документы    | `documents`       | Загрузка, версионирование, цикл согласования     |
| Чат          | `chat`            | Чат HR–кандидат, шаблонные опросы                 |
| Вакансии     | `vacancies`       | Реестр вакансий, отклики, собеседования, уведомления |
| Аналитика    | `analytics`       | Дашборд с графиками, отчёты в PDF/Excel          |

## Роли доступа

- **Кандидат** — вакансии, свои документы, чат с HR
- **HR-специалист** — управление вакансиями, документами, чаты с кандидатами
- **Руководитель** — всё то же + дашборд аналитики
- **Администратор** — всё то же + Django admin-панель (`/admin/`)

## Технологии

- Django 6.0.3, SQLite
- Bootstrap 5, HTMX, Chart.js (CDN)
